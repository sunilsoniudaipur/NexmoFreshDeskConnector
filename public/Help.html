Instructions

User Account setup and login

The connector supports multi-tenant configuration, which means that any user can create an account and configure the connector in the Freshdesk environment.

  1. To begin with configuration, a user needs to create an account using an email ID. Once the account is created, the user can login with the credentials.
  2. After successful login, user will need to make the necessary configurations for Freshdesk, Nexmo and the connector. The user needs to fill all the required details to enable connector.
  3. For each account, a unique configuration can be managed.

Available Commands

  1. New: To raise a new ticket request, the user can either text the command as “New” followed by the issue description or directly write a description for the issue. In case there is already an open ticket for the user, the connector will text back a menu with the opened tickets and options
  2. Open: To view all the open tickets, the user can text “OPEN” command. The connector will text back all open tickets in batches of three at a time.
  3. Reply: to reply to a ticket, the user will text REPLY followed by the response. This will notify the agent and add the response to the last seen ticket
  4. Help: To see all available commands supported by the connector, the user will text HELP command to list all supported commands with detail
